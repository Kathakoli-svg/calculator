# Smooth Calculator

A simple and smooth calculator built using **HTML, CSS, and JavaScript**.  
It performs basic arithmetic operations with a clean modern UI and smooth animations.

## Live Demo

    https://effulgent-platypus-e03593.netlify.app/

## Features

✅ Clean and modern UI  
✅ Smooth hover and click animations  
✅ Responsive layout  
✅ Basic arithmetic operations (+, −, ×, ÷)  
✅ Error handling for invalid input  
✅ Separate HTML, CSS, and JavaScript files
